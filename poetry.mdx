
This uses poetry install --without dev to skip development dependencies.

The poetry.Makefile has three install options:

1. install - Install all dependencies (dev + production)
2. install-no-dev - Install only production dependencies (excludes dev)
3. install-no-root - Install dependencies without installing the project itself
